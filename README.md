perf_monitoring_weighted_average_stats.py
    Breaks a csv file containing data with goal times and performance times into groups of some attribute and weights each group proportional to time donated.
    Helps us investigate if there might be unconsidered factors when calculating the difficulty of a work assighnment.
    Used to prove our workers were not responsible for a site wide loss in performance for a week in march of 2026.
